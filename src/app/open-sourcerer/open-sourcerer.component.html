I'm an open sourcerer

For years now, I'm convinced that open source is the good way to go!

I learned to code using open and free or affordable resources on the internet. In my day to day life I use hundred (if it's not even more) of open source projects to help me with my professional or personnal projects.

It is logical for me to give back to the open source community. Currently, I'm doing this in two different ways: creating learning ressources and contributing to the open source world.

I maintain some projects as @mindsers on Github. Here is a glance of them:

  • Changelog Reader Action

    A Github action to read and get data from the CHANGELOG.md file

  • Configfile

    Personal config files manager

  • Nativetable

    Simple native es6 module to create and work with dynamics HTML tables

  • YABF

    Yet Another Basic Framework for NodeJS (built for learning purpose)

  • Awwbar

    Awesome bended (circular) progress bar for iOS

I try to also take time to contribute to the open source projects I use. So I'm glad to be a contributor of dayzed, graphql-tag.macro and many others...
